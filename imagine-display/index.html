Oculus

The Oculus Rift is a head-mounted display created by Oculus VR company.

Virtual reality is a really cool concept, and the HMD puts us one step closer to true virtual reality. We aquired one earlier in the year for use in our projects. We've had a huge number of ideas on how to use it - today it's our 'virtual archer'.

Virtual Archer

Conceptually it's pretty simple, shoot arrows in a virtual world.

The control scheme was the hardest to settle on. We've implemented LEAP motion controls, but we've found the fidelity of the controls was not high enough for repeated motions. Until we've integrated something else cool, we have a MakeyMakey hooked up to an arcade button - we're shooting it old school.

Technology

The Virtual Archer works in the web browser! With a small browser plugin for the rift, everything else if pure HTML and Javascript, but we've pulled in quite a ton of open source code.

Bootstrap.js

A responsive UI framework for beautiful interfaces.

THREE.js

A wrapper around the WebGL and Canvas renderers, making 3D in the browser simpler.

Physi.js

A Threaded Physics engine, utilizing Ammo.js (a port of the Bullet physics engine)

Entity.js

Our own wrapper around THREE.js and Physi.js to create a simple environment for spawning interactive parts

A library of javascript convinience functions and DOM manipulators

vr.js

A wrapper around the oculus rift plugin that makes using the display simple.

An interface for LEAP motion controls

The Makey Makey reconfigurable improvisional controller - functions as a plug and play keyboard, no code needed.
